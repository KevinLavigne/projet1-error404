De quoi s'agit-il ?

Signification du code 404.

Il semblerait qu’il y est 3 explications possibles : * la première serait d’un point de vue technique. Le premier 4 du code signifie qu'une mauvaise adresse a été entrée. Le second fait référence au problème induit par cette mauvaise adresse. * la seconde explication voudrait que l’erreur 404 serait en réalité un hommage à Tim Berners-Lee qui occupait le bureau 404 au CERN. * la dernière explication, selon la légende, ferait aussi référence à ce même bureau 404. Mais cette fois, des ingénieurs agacés d’avoir à retourner dans ce fameux bureau pour relancer un serveur constamment planté auraient décidé de baptiser ainsi cette erreur courante. N’ayant à ce jour pas le moyen de verifier ces informations ou de preuves, son origine reste donc libre d’interprétation.


Qu’est ce que l’erreur 404

Il s’agit d’un code d’erreur HTTP transmis par un serveur web quand une ressource demandée est indisponible ou alors que le serveur n’arrive pas à la trouver. Le plus souvent, c’est une page web qui n’existe pas ou qui n’existe plus. Il n’est pas rare de trouver une erreur 404 sur un site et elle est facilement reconnaissable ; elle stoppe la navigation en cours avec un message d’erreur pour informer l’internaute, généralement : 404 page not find.


Ce code peut également être utilisé pour d’autres choses: censure une page existante, refonte d'un site internet, pour une page d'un site internet encore en construction


Les causes possibles :

* Un internaute fait une faute de frappe. * Une erreur de syntaxe dans la saisie d’une URL. * La suppression d’une page existante sans prise en compte des redirections comme après la mise en place d’une refonte de site Internet. * La modification de l’emplacement d’une page sur le site, ce qui peut modifier son URI (ajout ou suppression de répertoire dans l’url par exemple). * Un incident côté serveur qui génère de manière assez inhabituelle une page d’erreur 404.


Reparer une erreur 404

Réparer une erreur 404 : en insérant une nouvelle URL d’une nouvelle page à la place de l’URL défaillante. Autre possibilité : faire une redirection 301 : il s’agit d’une redirection permanente en indiquant que l’URL avec l’erreur qui doit être changée de manière permanente vers la nouvelle URL.

Est-elle vouée à disparaître ?

Corriger une erreur 404 est de nos jours facilement réalisable. De plus en plus de logiciels et d’ outils permettent de detecter et corriger ce type d’erreurs : comme notamment l’outil Search Console Google qui permet de detecter les erreurs 404 grace aux robots d’indexations de google. Apres connexion à votre compte la liste d’erreur est facilement accessible. Il existe également des outils d’exploration comme google et d’autres applications et extensions, de navigateurs qui détectent les erreurs 404 (aussi utiliser pour améliore le SEO) et peut les corriger/rediriger automatiquement.

Quelques exemples de messages d'erreur

Code erreur Message affiché Signification
ERREUR 302 Found Document déplacé de façon temporaire.
ERREUR 401 Authorized La requête nécessite une authentification de l’utilisateur.
ERREUR 403 Access denied Pas de possibilité de connexion disponible.
ERREUR 404 Not Found Le document appelé n’a pas été trouvé.
ERREUR 500 Internal Server Error Le serveur a rencontré une condition inattendue qui l'a empêché de répondre à la demande.
